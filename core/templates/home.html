"dj-indoor-monitor" es un sistema con licencia MIT diseñado para monitorizar y analizar datos ambientales en tiempo real. Proporciona información detallada sobre temperatura y humedad, recolectada desde sensores conectados a dispositivos Raspberry PI con diversos sensores.

La API de KCBD puede ser consumida desde aquí: 'https://kcbd.grupoalvs.com/timeframed/'.

Algunos parámetros opcionales son:

  • metadata: true o false
  • start_date: ej. 2025-01-01
  • end_date: si se prefiere una diferente a la fecha actual
  • timeframe: que acepta valores tales como '5s', '1D', etc.
